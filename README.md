## What this project does
This is an AI assistant that helps students with:
- Summaries
- Quiz generation
- Questions
- Study planning

## How to run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py

## Features
- AI summary generator
- Quiz maker
- Question generator
- Study planner
